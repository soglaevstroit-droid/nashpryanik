# packages/shared

Назначение: будущие общие типы, константы и утилиты, которые могут использоваться несколькими приложениями.

Источник истины:

- `DEVELOPMENT_RULES.md`
- `TECH_STACK_DECISION.md`
- `03_Database`
- `04_API`
- `09_Event_Bible`

На Sprint 001 здесь создается только место для shared package. Типы, enum, бизнес-правила и утилиты не создаются, чтобы не дублировать утвержденную документацию до начала инженерной реализации.
