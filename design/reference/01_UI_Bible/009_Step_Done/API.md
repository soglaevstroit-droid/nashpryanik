# API

## Общие требования
- Все запросы идут через текущий домен: `window.API_BASE = window.location.origin`.
- Все API начинаются с `/api`.
- Ошибки возвращаются в едином формате: `{ ok: false, error: "..." }`.
- Успех возвращается в формате: `{ ok: true, ... }`.

## Фото
Все фото отправляются через `multipart/form-data`.
Фото сохраняются в `server/uploads/photos/`.

## Экранные API
Уточнить конкретные endpoint по экрану в реализации. Использовать существующие маршруты, если они уже есть.

- `POST /api/tasks/:id/steps/:stepId/complete`
