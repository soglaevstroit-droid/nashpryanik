# LOGIC — 17 Создание задачи

## Основная логика
Прораб создает задачу: название, место, описание, исполнитель один или несколько, срок необязателен, этапы.

## Правила прораба
- Прораб управляет задачами и сотрудниками.
- Главный экран прораба начинается со списка сотрудников.
- Прораб видит кто на работе, кто ушел, кто чем занимается.
- Прораб может создать задачу.
- Исполнитель может быть один или несколько.
- Срок выполнения можно поставить или не ставить.
- Каждый этап содержит название, описание и необязательное фото-пример.
- Прораб может писать сотрудникам в чате.
- Прораб не начисляет монеты вручную.

## Создание задачи
Поля:
- название;
- место;
- описание;
- исполнитель: один / несколько / не выбран;
- срок: можно поставить / можно не ставить;
- этапы.

Если исполнитель не выбран — задача свободна.
Если выбран один исполнитель — взять может только он.
Если выбрано несколько — взять может любой из выбранных.
