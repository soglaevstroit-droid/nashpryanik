# UI — 04 Открытая задача

## Визуальный стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Inter / system sans-serif.
- Минимум визуального шума.
- iPhone-first.

## Обязательные элементы
Экран задачи: название, место, единая шкала этапов, текущий этап, описание, фото-пример, действия.

## Запрещено
- Добавлять лишние блоки.
- Менять композицию утвержденного референса.
- Показывать монтажнику тариф.
- Использовать разные иконки задач вместо одной фирменной.
- Дублировать прогресс этапов двумя шкалами.

## Компоненты
Использовать компоненты из `02_Design_System`.
