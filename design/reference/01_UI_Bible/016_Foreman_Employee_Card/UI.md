# UI — 16 Карточка сотрудника

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Детальная карточка сотрудника для прораба: статус, текущая работа, история, задачи, монеты, время.
