# VALIDATION — 25 Общий список задач

## Проверки
- Название задачи обязательно.
- Место задачи желательно, но MVP допускает пустое значение, если логика проекта это разрешает.
- Описание задачи желательно.
- Исполнитель может быть не выбран — тогда задача считается свободной.
- Исполнителей может быть несколько — задача доступна выбранным исполнителям.
- Срок выполнения необязателен.
- Этап должен иметь название или описание.
- Фото-пример этапа необязательно.
