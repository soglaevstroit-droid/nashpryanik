# 004 — Layout

## Общие правила
- Mobile First.
- Основная ширина проектируется под iPhone.
- Контент располагается в карточках.
- Отступы должны быть одинаковыми на всех экранах.

## Рекомендуемые отступы
- Screen horizontal padding: 16 px
- Card padding: 16 px
- Gap between cards: 12 px
- Section gap: 24 px
- Bottom navigation safe area: учитывать iPhone safe-area.

## Скругления
- Большие карточки: 20–24 px
- Малые карточки: 14–18 px
- Кнопки: 16–20 px
- Поля ввода: 14–18 px
