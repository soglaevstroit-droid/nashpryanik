# 002 — Colors

## Основная палитра
- Brand Orange: `#F07A00`
- Black: `#111111`
- White: `#FFFFFF`
- Light Background: `#F7F7F7`
- Card Border: `#E5E7EB`
- Text Gray: `#6B7280`
- Success Green: `#22C55E`
- Info Blue: `#3B82F6`
- Error Red: `#EF4444`
- Disabled Gray: `#D1D5DB`

## Правила
Оранжевый используется только для главного действия, фирменных иконок и ключевых акцентов.
Зеленый используется только для статуса успеха или «На работе».
Серый используется для недоступных задач и второстепенного текста.
Красный используется только для ошибок и опасных действий.
