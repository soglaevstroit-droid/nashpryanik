# 010 — Reference Rules

## Правило 95%
Codex обязан повторять утвержденные референсы минимум на 95%.

## Что считается нарушением
- изменение композиции экрана;
- замена карточек на другой стиль;
- добавление лишних блоков;
- изменение нижней навигации;
- изменение логики главной кнопки;
- изменение цветовой системы.

## Если референс и текстовая логика конфликтуют
Приоритет:
1. Утвержденная логика.
2. Утвержденный референс.
3. Design System.
4. Текущее решение разработчика.
