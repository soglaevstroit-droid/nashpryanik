# 005 — Buttons

## Primary Button
Используется для главного действия экрана.

Примеры:
- Войти
- Сделать фото
- Взяться за работу
- Сделал?
- Сохранить

Стиль:
- фон: Brand Orange
- текст: White
- высота: 52–56 px
- радиус: 16–20 px

## Secondary Button
Для отмены или дополнительного действия.

## Danger Button
Для отказа или опасного действия:
- Не вывожу
- Удалить
- Отменить

## Правило
На экране должна быть только одна визуально главная кнопка.
