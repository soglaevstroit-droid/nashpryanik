# ADR-002

## Название

Single Server Infrastructure for MVP

---

Статус

✅ APPROVED

---

Дата

_____________

---

Контекст

На первом этапе проект запускается на одном объекте.

Количество пользователей:

до 10 сотрудников.

Необходимо выбрать инфраструктуру.

---

Рассматривались варианты

1. Kubernetes

2. Несколько серверов

3. Один сервер

---

Принятое решение

Использовать один VPS.

Характеристики

- Ubuntu 24.04 LTS
- 4 vCPU
- 8 GB RAM
- 200 GB NVMe

Docker:

- PostgreSQL
- Redis
- Backend
- MinIO
- Nginx

SSL

Let's Encrypt

---

Причины

- минимальная стоимость;
- простота обслуживания;
- быстрый запуск;
- достаточно для MVP;
- возможность масштабирования без изменения архитектуры приложения.

---

Последствия

Положительные

- дешево;
- быстро;
- надежно;
- простая поддержка.

Отрицательные

- одна точка отказа;
- при росте потребуется разделение сервисов.

---

Будущее

После достижения:

- 100 пользователей

или

- нескольких объектов

необходимо перейти к горизонтальному масштабированию.
