# ADR-001

## Название

Technology Stack

---

Статус:

✅ APPROVED

---

Дата:

_____________

---

Контекст

Необходимо выбрать единый технологический стек проекта.

От выбора технологий зависит архитектура всей системы.

---

Решение

Утвержден следующий стек:

- Flutter

- NestJS

- PostgreSQL

- Prisma

- Redis

- BullMQ

- JWT

- MinIO

- Docker

- Docker Compose

- Nginx

- GitHub Actions

---

Причины выбора

- надежность;

- масштабируемость;

- большое сообщество;

- современная архитектура;

- простота сопровождения;

- высокая производительность.

---

Последствия

Положительные

- единая архитектура;

- быстрое развитие;

- простое масштабирование.

Отрицательные

- изменение технологий возможно только после создания нового ADR.
