# 008 — Icons

## Главное правило
Иконки помогают быстрее понять интерфейс, но не являются украшением.

## Задачи
Для задач используется одна единая фирменная иконка оранжевого цвета.

Если задачу нельзя взять:
- иконка и карточка становятся черно-белыми / приглушенными.

## Запрещено
- использовать разные иконки задач по типам работ в MVP;
- использовать декоративные иконки без смысла.
