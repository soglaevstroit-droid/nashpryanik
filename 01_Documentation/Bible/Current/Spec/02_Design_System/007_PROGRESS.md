# 007 — Progress

## Единая шкала этапов
Используется одна понятная шкала без дублей.

Запрещено одновременно показывать:
- верхнюю шкалу этапов;
- нижний прогресс;
- отдельный список статусов с тем же смыслом.

## Цвета
- Выполнено: Green
- Текущий этап: Orange
- Будущее: Light Gray

## Формат
Под шкалой допускается короткий текст: `3 из 8 этапов`.
Без лишних иконок и дублирующих подписей.
