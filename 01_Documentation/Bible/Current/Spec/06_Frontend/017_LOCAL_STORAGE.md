# 017 — Local Storage

## Назначение
Документ описывает локальное хранение данных Flutter-приложения.

## Secure storage
Хранит:
- refresh token;
- access token, если выбран secure storage;
- session metadata.

Запрещено хранить token в plain preferences.

## Cache storage
Хранит:
- последние данные стартового экрана;
- список задач;
- список сотрудников;
- фильтры;
- справочники объектов;
- уведомления;
- черновики форм без финансового утверждения.

## Upload queue
Хранит:
- local file path;
- photo type;
- related ids;
- retry count;
- upload status.

## Sync queue
Хранит безопасные pending actions. Финансовые действия не должны повторяться без idempotency и обработки конфликтов.

## Settings
Локально можно хранить UI-предпочтения, которые не противоречат backend:
- последний открытый tab;
- выбранный период;
- локальный cache timestamp.

Финансовые настройки из `040_Finance_Settings` должны сохраняться через API.

## Очистка
При logout:
- удаляются token;
- очищается role state;
- останавливаются защищенные sync операции;
- чувствительный cache очищается или помечается недоступным.

## Ограничение
Local storage не является источником истины для монет, выплат, AI-рекомендаций и статусов задач.
