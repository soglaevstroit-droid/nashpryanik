# Общая философия

Мы выбираем не самые модные технологии.

Мы выбираем технологии, которые:

- надежны;
- имеют большое сообщество;
- хорошо документированы;
- подходят для долгосрочного развития;
- легко масштабируются;
- позволяют быстро обучать новых разработчиков.

# Основные принципы

1. Простота важнее сложности.
2. Читаемость важнее сокращения кода.
3. Масштабируемость важнее локальной оптимизации.
4. Документация является частью продукта.
5. Код обязан соответствовать документации.
6. Архитектура важнее скорости разработки.
7. Безопасность обязательна.
8. Все изменения проходят Code Review.
9. AI помогает человеку, но не принимает бизнес-решения.
10. Любая технология должна служить людям, а не наоборот.
