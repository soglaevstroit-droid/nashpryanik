# 04 API

## Назначение
Раздел `04_API` описывает REST API проекта НАШПРЯНИК.РФ. Все endpoint-ы используют префикс `/api/v1`, JWT-аутентификацию, role-based access control и единый формат ошибок.

API-документация связывает UI Bible, Architecture Bible и PostgreSQL-модель. Каждый endpoint должен соответствовать утвержденной бизнес-логике: монтажник выполняет задачи и подтверждает действия фото, прораб управляет задачами и сотрудниками, финансист анализирует монеты, AI-рекомендации и принимает решения по выплатам.

## Общие правила
- Все endpoint-ы начинаются с `/api/v1`.
- Все защищенные запросы используют `Authorization: Bearer <jwt>`.
- Сервер проверяет роль независимо от UI.
- Фото отправляются через `multipart/form-data`.
- Финансовые действия пишутся в `audit_log`.
- ИИ только рекомендует, человек утверждает.
