# MASTER PROMPT ДЛЯ CODEX

Проект: НАШПРЯНИК.РФ  
Версия: v1.0

## Главная задача

Реализовать приложение по утвержденным референсам и логике.

## Дизайн

Повторить утвержденные референсы минимум на 95%.

Нельзя:
- менять стиль;
- добавлять лишние элементы;
- усложнять интерфейс;
- менять логику экранов;
- показывать монтажнику лишнюю аналитику.

## Главная логика

Монтажник:
- видит общий список задач;
- может взять свободную задачу;
- может выполнять только одну задачу одновременно;
- все ключевые действия подтверждаются фото;
- монеты начисляются только в статусе На работе.

Прораб:
- видит сотрудников;
- создает задачи;
- назначает одного или нескольких исполнителей;
- создает этапы с описанием и фото-примером.

Финансист:
- видит заработанные сотрудниками монеты;
- видит мнение ИИ о справедливой стоимости работы;
- видит рекомендованные премии;
- принимает финальное решение по выплатам.

## Правило

ИИ рекомендует. Человек утверждает.
