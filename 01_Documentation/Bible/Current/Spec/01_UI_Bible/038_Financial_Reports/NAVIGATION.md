# NAVIGATION — 38 Финансовые отчеты

## Вход на экран
Открывается из аналитики финансиста, истории выплат или нижнего меню финансового раздела.

## Выход
- Назад к аналитике.
- Открыть карточку сотрудника.
- Открыть историю выплат.
- Открыть настройки финансиста.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Нажатие на сотрудника открывает карточку сотрудника.
- Нажатие на строку отчета открывает детализацию периода.
- Выгрузка остается действием текущего экрана.
