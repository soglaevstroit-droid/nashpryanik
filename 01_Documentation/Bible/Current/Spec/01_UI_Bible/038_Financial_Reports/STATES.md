# STATES — 38 Финансовые отчеты

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Exporting
- Export ready
- Export error
- Disabled action

## Loading
Показываются скелетоны фильтров, сводных карточек и таблицы.

## Empty
Показывается состояние «Нет данных для отчета за выбранный период».

## Exporting
Показывается во время подготовки файла выгрузки.

## Export ready
Показывается, когда файл отчета готов к скачиванию.

## Error
Показывается ошибка загрузки или формирования отчета.

## Правила
Состояния выгрузки не должны менять данные отчета на экране.
