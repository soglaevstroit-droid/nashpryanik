# VALIDATION — 38 Финансовые отчеты

## Проверки
- `periodFrom` не может быть позже `periodTo`.
- `objectId` должен существовать, если передан.
- `employeeId` должен существовать, если передан.
- Формат выгрузки должен быть поддержан.
- Начисленные монеты не могут быть отрицательными.
- Премии не могут быть отрицательными.
- Итог к выплате должен равняться начисленным монетам плюс премии.
- Строки отчета должны соответствовать текущим фильтрам.

## Отображение
- Суммы отображаются целым числом.
- Период отображается понятным диапазоном дат.
- Выгрузка недоступна при ошибке загрузки данных.
