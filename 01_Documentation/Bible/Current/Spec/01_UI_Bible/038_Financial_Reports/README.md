# 38 — Финансовые отчеты

## Назначение
Экран финансовых отчетов по периодам, объектам, сотрудникам, начисленным монетам, премиям, итогам к выплате и рекомендациям ИИ.

## Роль
Финансист / отчеты.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Отчет фиксирует и выгружает данные, но не меняет начисления и не утверждает выплаты автоматически.
