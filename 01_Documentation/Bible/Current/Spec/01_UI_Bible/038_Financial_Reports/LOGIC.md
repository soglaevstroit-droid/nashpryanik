# LOGIC — 38 Финансовые отчеты

## Основная логика
Экран формирует финансовый отчет по выбранному периоду, объектам и сотрудникам.

## Данные отчета
- Период задает границы расчета.
- Объект ограничивает данные конкретной строительной площадкой или объектом.
- Сотрудники ограничивают строки отчета выбранными людьми.
- Начисленные монеты берутся из подтвержденных начислений.
- Премии берутся из утвержденных решений финансиста.
- Итог к выплате равен начисленным монетам плюс премии.
- Рекомендации ИИ показываются справочно.

## Выгрузка отчета
- Выгрузка строится по текущим фильтрам.
- Выгрузка не меняет статус выплат.
- Если данных нет, выгрузка недоступна или формирует пустой отчет с заголовками.

## Рекомендации ИИ
- ИИ может объяснить расхождения и предложить обратить внимание на сотрудника.
- ИИ не утверждает выплату.
- ИИ не меняет итог к выплате без решения финансиста.

## Историчность
Если отчет строится по уже утвержденным выплатам, значения должны соответствовать истории выплат.
