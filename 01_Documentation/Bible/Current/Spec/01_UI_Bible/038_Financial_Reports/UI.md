# UI — 38 Финансовые отчеты

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Финансовые отчеты показывают сводку выплат и рекомендаций за выбранный период.

## Основная композиция
- Верхняя зона: заголовок «Финансовые отчеты».
- Фильтры: период, объект, сотрудники.
- Сводные карточки: начисленные монеты, премии, итог к выплате.
- Таблица сотрудников.
- Колонка рекомендаций ИИ.
- Действие выгрузки отчета.

## Таблица отчета
- Строка содержит сотрудника, объект, период, начисленные монеты, премии, итог к выплате и AI рекомендацию.
- Итоговые значения закрепляются внизу или в сводном блоке.
- Выгрузка доступна только для текущих фильтров.

## Запрещено
- менять данные отчета вручную;
- использовать выгрузку как подтверждение выплаты без решения финансиста;
- создавать новые стили таблиц;
- использовать другие цвета.
