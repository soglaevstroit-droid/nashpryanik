# STATES — 31 Аналитика сотрудника

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Employee not found
- Disabled action

## Loading
Показываются скелетоны карточек, диаграммы и истории активности.

## Empty
Показывается состояние без активности за выбранный период.

## Employee not found
Показывается, если сотрудник удален или недоступен текущей роли.

## Error
Показывается ошибка загрузки с возможностью повторить запрос.

## Правила
Состояния не должны менять порядок основных блоков экрана.
