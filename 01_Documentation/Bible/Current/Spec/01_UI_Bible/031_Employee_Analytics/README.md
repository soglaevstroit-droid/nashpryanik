# 31 — Аналитика сотрудника

## Назначение
Экран детальной аналитики одного сотрудника: монеты, рабочее время, выполненные задачи, история активности и диаграмма работы.

## Роль
Финансист / финансовая аналитика.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Экран объясняет вклад сотрудника через подтвержденные данные и не меняет начисления.
