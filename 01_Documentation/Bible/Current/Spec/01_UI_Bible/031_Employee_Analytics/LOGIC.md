# LOGIC — 31 Аналитика сотрудника

## Основная логика
Экран показывает аналитику выбранного сотрудника за выбранный период.

## Показатели
- Монеты — сумма подтвержденных начислений сотрудника.
- Рабочее время — сумма подтвержденных рабочих смен сотрудника.
- Выполненные задачи — количество задач, закрытых сотрудником или с его участием.
- История активности — последовательность подтвержденных рабочих событий.
- Диаграмма работы — визуальное распределение работы по дням или задачам.

## Выполненные задачи
- Задача учитывается, если она имеет статус завершения.
- Если задача выполнялась несколькими сотрудниками, вклад сотрудника учитывается по правилам начисления монет.
- Отмененные задачи не входят в выполненные.

## История активности
- Показываются только события, доступные роли финансиста.
- Исторические данные сохраняются даже при архивировании объекта или задачи.
- Смена периода обновляет все показатели сотрудника.

## Правила финансиста
- Финансист анализирует данные сотрудника.
- Финансист может перейти к AI рекомендации по сотруднику.
- Финансист не принимает автоматическое решение на этом экране.
