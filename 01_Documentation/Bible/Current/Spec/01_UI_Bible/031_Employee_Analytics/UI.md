# UI — 31 Аналитика сотрудника

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Аналитика сотрудника показывает персональные финансовые и рабочие показатели.

## Основная композиция
- Верхняя зона: имя сотрудника, объект или текущий объект, статус.
- Карточка «Монеты».
- Карточка «Рабочее время».
- Карточка «Выполненные задачи».
- Диаграмма работы за выбранный период.
- История активности.

## История активности
- События показываются в обратном хронологическом порядке.
- Событие содержит дату, объект, задачу, тип активности, монеты и рабочее время.
- Визуальный акцент ставится на подтвержденные события.

## Запрещено
- редактировать данные сотрудника с этого экрана;
- вручную менять монеты;
- смешивать историю активности с личными данными, не связанными с работой;
- скрывать источник данных по задачам и сменам.
