# VALIDATION — 29 Главная финансиста

## Проверки
- `totalCoins` обязателен и не может быть отрицательным.
- `workingEmployeesCount` обязателен и не может быть отрицательным.
- Количество работающих сотрудников должно соответствовать списку активных смен.
- `employeeId` обязателен для перехода в аналитику сотрудника.
- `totalCoins` сотрудника не может быть отрицательным.
- Длительность текущей смены не может быть отрицательной.

## Отображение чисел
- Монеты отображаются целым числом.
- Большие числа форматируются с разделителями разрядов.
- При отсутствии данных показывается `0`, а не пустое значение.
