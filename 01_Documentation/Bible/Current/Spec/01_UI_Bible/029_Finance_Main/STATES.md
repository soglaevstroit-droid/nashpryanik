# STATES — 29 Главная финансиста

## Состояния
- Loading
- Counting animation
- Ready
- Empty
- Error
- Offline
- Refreshing
- Disabled action

## Loading
Показывается скелетон главного числа и списка сотрудников.

## Empty
Показывается нулевая сумма монет и текстовое состояние без сотрудников.

## Error
Показывается сообщение об ошибке загрузки финансовых данных и действие повтора.

## Offline
Показываются последние сохраненные данные, если они доступны, с отметкой о возможной неактуальности.

## Правила
Состояния не должны ломать композицию экрана и размер главного числового блока.
