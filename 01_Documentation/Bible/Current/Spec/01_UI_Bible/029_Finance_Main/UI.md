# UI — 29 Главная финансиста

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Главная финансиста открывается с крупного числового блока «Всего собрано монет».

## Основная композиция
- Верхняя зона: заголовок «Финансы» и короткий период «За все время».
- Главный блок: очень крупная анимированная цифра суммы монет.
- Подпись под цифрой: «Всего собрано монет».
- Вторичный блок: количество сотрудников, которые сейчас работают.
- Список сотрудников: имя, объект, статус, монеты за выбранный контекст, текущее рабочее время.

## Анимация числа
- При загрузке значение плавно увеличивается от 0 до фактической суммы.
- Анимация не меняет итоговое значение.
- При обновлении данных число мягко пересчитывается без скачка верстки.

## Запрещено
- скрывать главный показатель ниже первого экрана;
- смешивать финансовую аналитику с управлением задачами;
- добавлять ручное начисление монет;
- менять нижнюю навигацию без причины;
- использовать разные стили карточек.
