# NAVIGATION — 29 Главная финансиста

## Вход на экран
Открывается после авторизации пользователя с ролью финансиста.

## Выход
- Открыть общую аналитику.
- Открыть аналитику сотрудника.
- Открыть AI рекомендации.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Рекомендации
- Профиль

## Переходы
- Нажатие на главный показатель открывает общую аналитику.
- Нажатие на сотрудника открывает аналитику сотрудника.
- Нажатие на блок рекомендации открывает AI рекомендации.
