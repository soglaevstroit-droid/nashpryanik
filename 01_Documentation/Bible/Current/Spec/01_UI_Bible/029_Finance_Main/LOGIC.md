# LOGIC — 29 Главная финансиста

## Основная логика
Экран показывает агрегированную сумму монет, собранных по всем объектам, всем сотрудникам и за все время.

## Расчет суммы
- Сумма строится только по подтвержденным начислениям монет.
- Черновики, отмененные задачи и неподтвержденные начисления не входят в расчет.
- Если сотрудник работал на нескольких объектах, его монеты учитываются один раз в общей сумме.
- Значение не зависит от текущего фильтра списка сотрудников.

## Работающие сотрудники
- Сотрудник считается работающим, если у него есть активная рабочая смена.
- Сотрудник отображается в списке даже при нулевых монетах, если он сейчас на работе.
- Статус «На работе» не означает право на выплату, а только фиксирует активность.

## Список сотрудников
- По умолчанию первыми отображаются сотрудники со статусом «На работе».
- Далее сотрудники сортируются по количеству монет за все время по убыванию.
- Нажатие на сотрудника открывает экран аналитики сотрудника.

## Правила финансиста
- Финансист видит финансовую картину и аналитику.
- Финансист не создает задачи на этом экране.
- Финансист не меняет начисленные монеты вручную.
- Финансист принимает финальное решение по премиям только после просмотра рекомендации и данных.
