# STATES — 32 AI рекомендации

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Saving decision
- Decision saved
- Disabled action

## Loading
Показываются скелетоны карточек рекомендаций.

## Empty
Показывается состояние «Нет рекомендаций по премии за выбранный период».

## Saving decision
Показывается только при сохранении решения финансиста, если оно предусмотрено бизнес-процессом.

## Decision saved
Показывается подтверждение, что решение финансиста сохранено.

## Error
Показывается ошибка загрузки или сохранения с возможностью повторить действие.

## Правила
Состояния не должны превращать рекомендацию ИИ в автоматическое решение.
