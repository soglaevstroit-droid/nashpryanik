# NAVIGATION — 32 AI рекомендации

## Вход на экран
Открывается из нижнего меню финансиста, главной финансиста, общей аналитики или аналитики сотрудника.

## Выход
- Назад к предыдущему экрану.
- Открыть аналитику сотрудника.
- Открыть общую аналитику.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Рекомендации
- Профиль

## Переходы
- Нажатие на сотрудника открывает аналитику сотрудника.
- Нажатие на рекомендацию открывает детальное объяснение.
- Сохранение решения оставляет пользователя на экране рекомендаций.
