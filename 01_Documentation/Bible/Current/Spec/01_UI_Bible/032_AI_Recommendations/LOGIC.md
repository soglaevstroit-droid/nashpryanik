# LOGIC — 32 AI рекомендации

## Основная логика
ИИ сравнивает начисленные монеты с собственной оценкой стоимости выполненной работы.

## Расчет рекомендации
- `paidCoins` — подтвержденные начисленные монеты сотрудника за период.
- `aiEstimatedCoins` — оценка ИИ по стоимости выполненной работы за тот же период.
- `difference` = `aiEstimatedCoins` - `paidCoins`.
- Если `difference` больше 0, показывается рекомендация выплатить премию.
- Если `difference` меньше или равна 0, рекомендация по премии не показывается.

## Источники оценки ИИ
- Выполненные задачи.
- Сложность задач, если она доступна в данных.
- Рабочее время.
- История активности.
- Объект и контекст выполнения работ.

## Ограничения
- ИИ не меняет начисленные монеты.
- ИИ не создает выплату.
- ИИ не принимает финальное решение.
- Финансист может принять, отклонить или проигнорировать рекомендацию согласно бизнес-процессу выплат.

## Объяснимость
Рекомендация должна показывать краткое основание: какие данные повлияли на оценку и почему оценка выше начисленных монет.
