# VALIDATION — 32 AI рекомендации

## Проверки
- `paidCoins` обязателен и не может быть отрицательным.
- `aiEstimatedCoins` обязателен и не может быть отрицательным.
- `difference` должен равняться `aiEstimatedCoins - paidCoins`.
- Рекомендация премии показывается только при `difference > 0`.
- `employeeId` обязателен для персональной рекомендации.
- `confidence` должен быть в диапазоне от 0 до 1, если поле используется.
- Решение финансиста должно быть одним из допустимых значений.
- Комментарий финансиста не должен превышать лимит, заданный API.

## Отображение
- Монеты отображаются целым числом.
- Положительная разница отображается как потенциальная премия.
- Нулевая или отрицательная разница не должна выглядеть как долг сотрудника.
