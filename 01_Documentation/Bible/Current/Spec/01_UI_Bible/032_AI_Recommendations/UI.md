# UI — 32 AI рекомендации

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
AI рекомендации показывают сравнение начисленных монет и оценки ИИ по стоимости выполненной работы.

## Основная композиция
- Верхняя зона: заголовок «AI рекомендации» и выбранный период.
- Карточка сотрудника или список сотрудников с рекомендациями.
- Блок «Начисленные монеты».
- Блок «Оценка ИИ».
- Разница между оценкой ИИ и начисленными монетами.
- Рекомендация по премии, если оценка ИИ выше начисления.
- Дисклеймер: «ИИ рекомендует. Решение принимает финансист».

## Рекомендация
- Если оценка ИИ выше начисленных монет, показывается рекомендация выплатить премию.
- Если оценка ИИ равна или ниже начисленных монет, показывается нейтральное состояние без премии.
- Рекомендация должна быть визуально отделена от финального решения.

## Запрещено
- автоматически назначать премию;
- показывать рекомендацию как обязательное решение;
- скрывать основание сравнения;
- заменять решение финансиста решением ИИ.
