# NAVIGATION — 39 Уведомления финансиста

## Вход на экран
Открывается из нижнего меню, главной финансиста или системного индикатора уведомлений.

## Выход
- Назад на главную финансиста.
- Открыть AI рекомендации.
- Открыть аналитику эффективности.
- Открыть выплаты.
- Открыть карточку сотрудника.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Уведомление о рекомендации открывает AI рекомендации.
- Уведомление об отклонении открывает аналитику эффективности.
- Уведомление о готовой выплате открывает экран выплат.
- Уведомление о сотруднике открывает карточку сотрудника.
