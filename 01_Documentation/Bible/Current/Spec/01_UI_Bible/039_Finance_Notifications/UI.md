# UI — 39 Уведомления финансиста

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Уведомления финансиста показывают события, требующие внимания.

## Основная композиция
- Верхняя зона: заголовок «Уведомления».
- Фильтры: тип, статус, период, объект.
- Список уведомлений.
- Счетчики новых и важных уведомлений.
- Карточка уведомления с типом, сотрудником или объектом, кратким описанием и датой.
- Действие перехода к связанному экрану.

## Типы уведомлений
- Новые рекомендации ИИ.
- Большие отклонения.
- Сотрудники без активности.
- Подозрительные действия.
- Готовые выплаты.
- События, требующие внимания.

## Запрещено
- выполнять выплату прямо из уведомления без экрана выплат;
- автоматически принимать рекомендацию ИИ;
- создавать новые цвета для важности;
- скрывать источник события.
