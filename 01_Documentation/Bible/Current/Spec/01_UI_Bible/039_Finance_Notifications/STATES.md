# STATES — 39 Уведомления финансиста

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Marking read
- Resolving
- Disabled action

## Loading
Показываются скелетоны фильтров и списка уведомлений.

## Empty
Показывается состояние «Нет уведомлений».

## Marking read
Показывается при отметке уведомления как просмотренного.

## Resolving
Показывается при отметке уведомления как обработанного.

## Error
Показывается ошибка загрузки или изменения статуса уведомления.

## Правила
Состояния не должны запускать выплаты или принимать рекомендации автоматически.
