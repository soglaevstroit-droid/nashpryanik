# 39 — Уведомления финансиста

## Назначение
Экран уведомлений финансиста о новых рекомендациях ИИ, отклонениях, сотрудниках без активности, подозрительных действиях, готовых выплатах и событиях, требующих внимания.

## Роль
Финансист / уведомления.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Уведомления помогают обратить внимание на событие, но не выполняют финансовые действия автоматически.
