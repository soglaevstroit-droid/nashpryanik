# LOGIC — 39 Уведомления финансиста

## Основная логика
Экран показывает финансовые и аналитические события, требующие внимания финансиста.

## Источники уведомлений
- Новые рекомендации ИИ.
- Большие отклонения эффективности.
- Сотрудники без активности за заданный период.
- Подозрительные действия в задачах, сменах или начислениях.
- Готовые выплаты.
- События, требующие внимания по правилам проекта.

## Статусы уведомлений
- Новое.
- Просмотрено.
- Обработано.
- Скрыто.

## Обработка
- Просмотр уведомления не выполняет финансовое действие.
- Переход из уведомления открывает связанный экран: рекомендацию, сотрудника, выплату, отчет или историю.
- Отметка «обработано» фиксирует действие финансиста, но не меняет начисления.

## Ограничения ИИ
- ИИ может быть источником рекомендации или объяснения.
- ИИ не утверждает выплаты.
- ИИ не скрывает уведомления автоматически без правила продукта.
