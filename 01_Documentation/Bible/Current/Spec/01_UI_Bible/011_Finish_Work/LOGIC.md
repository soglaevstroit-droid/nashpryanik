# LOGIC — 11 Уход с работы

## Основная логика
Подтверждение ухода, фотофиксация, остановка таймера и начисления монет.

## Бизнес-правила монтажника
- Монтажник может быть в статусе `На работе` или `Ушел`.
- Монеты начисляются только в статусе `На работе`.
- Монтажник видит общий список задач объекта.
- Свободную задачу может взять любой монтажник.
- Если задача назначена конкретному сотруднику — взять может только он.
- Если задача уже в работе — остальные сотрудники могут только просматривать.
- Одновременно у монтажника может быть только одна задача в работе.
- Все ключевые действия требуют фото.

## Ключевые события
- `work_started`
- `work_finished`
- `task_taken`
- `task_released`
- `step_completed`
- `task_completed`
