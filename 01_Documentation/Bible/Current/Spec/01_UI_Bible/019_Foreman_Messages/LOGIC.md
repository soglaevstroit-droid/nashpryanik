# LOGIC — 19 Сообщения прораба

## Основная логика
Список диалогов со всеми сотрудниками и чат с выбранным сотрудником.

## Правила прораба
- Прораб управляет задачами и сотрудниками.
- Главный экран прораба начинается со списка сотрудников.
- Прораб видит кто на работе, кто ушел, кто чем занимается.
- Прораб может создать задачу.
- Исполнитель может быть один или несколько.
- Срок выполнения можно поставить или не ставить.
- Каждый этап содержит название, описание и необязательное фото-пример.
- Прораб может писать сотрудникам в чате.
- Прораб не начисляет монеты вручную.
