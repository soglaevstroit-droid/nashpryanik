# LOGIC — 01 Моя работа / Главная

## Основная логика
Главный экран монтажника: имя, статус На работе/Ушел, баланс монет, таймер, общий список задач объекта.

## Бизнес-правила монтажника
- Монтажник может быть в статусе `На работе` или `Ушел`.
- Монеты начисляются только в статусе `На работе`.
- Монтажник видит общий список задач объекта.
- Свободную задачу может взять любой монтажник.
- Если задача назначена конкретному сотруднику — взять может только он.
- Если задача уже в работе — остальные сотрудники могут только просматривать.
- Одновременно у монтажника может быть только одна задача в работе.
- Все ключевые действия требуют фото.

## Ключевые события
- `work_started`
- `work_finished`
- `task_taken`
- `task_released`
- `step_completed`
- `task_completed`

## Уточненная логика главной монтажника
- Показывать только имя сотрудника, без должности.
- Верх: статус, баланс монет, таймер смены.
- Тариф не показывать.
- Список задач общий для всех сотрудников.
- Все задачи имеют одну фирменную оранжевую иконку.
- Если сотрудник уже взял задачу, остальные задачи приглушены и доступны только для просмотра.
