# UI — 34 Карточка сотрудника

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Карточка сотрудника показывает полную финансовую и рабочую картину по одному сотруднику.

## Основная композиция
- Верхняя зона: фото, имя, текущий статус.
- Блок объектов сотрудника.
- Карточка рабочего времени.
- Карточка заработанных монет.
- Карточка выполненных задач.
- Блок текущей задачи.
- Блок рекомендаций ИИ.
- Календарь активности.
- История активности.

## Календарь активности
- Активные дни отмечаются в рамках Design System.
- День открывает список событий за дату.
- Календарь не используется для ручного редактирования смен.

## История
- История отображается в обратном хронологическом порядке.
- Событие показывает объект, задачу, время, монеты и статус подтверждения.

## Запрещено
- редактировать профиль сотрудника;
- вручную менять монеты;
- выполнять выплату с карточки без перехода в экран выплат;
- показывать AI рекомендацию как обязательное решение.
