# LOGIC — 34 Карточка сотрудника

## Основная логика
Экран собирает данные сотрудника из рабочих смен, задач, начисленных монет, текущей активности и AI рекомендаций.

## Данные сотрудника
- Имя и фото берутся из профиля сотрудника.
- Текущий статус берется из активной смены.
- Объекты берутся из задач и назначений сотрудника.
- Рабочее время считается по подтвержденным сменам.
- Заработанные монеты считаются по подтвержденным начислениям.
- Выполненные задачи считаются по завершенным задачам с участием сотрудника.

## Текущая задача
- Показывается, если сотрудник сейчас выполняет задачу.
- Если активной задачи нет, показывается нейтральное состояние.
- Текущая задача не редактируется финансистом.

## AI рекомендации
- Показываются рекомендации, связанные с сотрудником и выбранным периодом.
- Рекомендация не изменяет сумму выплаты автоматически.
- Финальное решение остается за финансистом.

## Календарь и история
- Календарь показывает активность по дням.
- История показывает события: смены, задачи, начисления, рекомендации и выплаты, если они доступны роли финансиста.
- Смена периода обновляет календарь, историю и финансовые показатели.
