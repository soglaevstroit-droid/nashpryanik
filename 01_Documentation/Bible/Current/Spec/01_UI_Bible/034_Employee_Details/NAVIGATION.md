# NAVIGATION — 34 Карточка сотрудника

## Вход на экран
Открывается из главной финансиста, общей аналитики, диаграммы Ганта, истории выплат или AI рекомендаций.

## Выход
- Назад к предыдущему экрану.
- Открыть выплаты по сотруднику.
- Открыть историю выплат.
- Открыть диаграмму Ганта.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Нажатие на текущую задачу открывает задачу в режиме просмотра.
- Нажатие на рекомендацию ИИ открывает экран выплат или детальное объяснение рекомендации.
- Нажатие на день календаря фильтрует историю по дате.
