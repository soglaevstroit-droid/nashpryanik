# STATES — 34 Карточка сотрудника

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Employee not found
- No current task
- No AI recommendations
- Disabled action

## Loading
Показываются скелетоны фото, карточек, календаря и истории.

## Employee not found
Показывается, если сотрудник удален или недоступен финансисту.

## No current task
Показывается нейтральный блок без активной задачи.

## No AI recommendations
Показывается, если за выбранный период нет рекомендаций.

## Error
Показывается ошибка загрузки с действием повтора.

## Правила
Состояния не должны менять порядок основных блоков карточки сотрудника.
