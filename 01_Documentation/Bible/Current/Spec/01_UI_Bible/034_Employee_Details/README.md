# 34 — Карточка сотрудника

## Назначение
Детальная карточка сотрудника для финансиста с данными о статусе, объектах, времени, монетах, задачах, AI рекомендациях, календаре активности и истории.

## Роль
Финансист / финансовая аналитика.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Карточка сотрудника показывает подтвержденные рабочие и финансовые данные без редактирования начислений.
