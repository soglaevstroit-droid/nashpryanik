# LOGIC — 30 Общая аналитика

## Основная логика
Экран строит агрегированную аналитику по выбранному периоду.

## Показатели
- Суммарные монеты — сумма подтвержденных начислений за период.
- Суммарное рабочее время — сумма подтвержденных рабочих смен за период.
- Динамика по дням — распределение монет и времени по календарным дням.
- Динамика по объектам — распределение монет и времени по объектам.
- Динамика по сотрудникам — распределение монет и времени по сотрудникам.

## Период
- По умолчанию используется период, принятый для финансового раздела.
- Если период не выбран, показывается значение «За все время».
- Смена периода пересчитывает все диаграммы и агрегаты.

## Сопоставление данных
- Монеты связываются с сотрудником, задачей и объектом.
- Рабочее время связывается с подтвержденной сменой.
- Если объект удален или архивирован, его исторические данные остаются в аналитике.

## Правила финансиста
- Финансист может анализировать данные, но не редактирует начисления на этом экране.
- Переход в сотрудника сохраняет выбранный период как контекст.
