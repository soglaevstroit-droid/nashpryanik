# 30 — Общая аналитика

## Назначение
Экран общей финансовой аналитики по монетам, рабочему времени, дням, объектам и сотрудникам.

## Роль
Финансист / финансовая аналитика.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Экран показывает агрегаты и динамику, но не меняет начисления и не принимает решений вместо финансиста.
