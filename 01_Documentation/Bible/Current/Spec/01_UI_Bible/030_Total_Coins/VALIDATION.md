# VALIDATION — 30 Общая аналитика

## Проверки
- `periodFrom` не может быть позже `periodTo`.
- Период не должен выходить за границы доступной истории.
- Суммарные монеты не могут быть отрицательными.
- Суммарное рабочее время не может быть отрицательным.
- Значения по дням должны суммироваться в общий итог периода.
- Значения по объектам должны суммироваться в общий итог периода.
- Значения по сотрудникам должны суммироваться в общий итог периода.

## Отображение
- Монеты отображаются целым числом.
- Рабочее время отображается в часах и минутах.
- Пустые диаграммы показывают нулевое состояние, а не ошибку.
