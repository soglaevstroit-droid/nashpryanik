# STATES — 30 Общая аналитика

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Filter changed
- Disabled action

## Loading
Показываются скелетоны карточек и диаграмм.

## Empty
Показывается нулевая аналитика за выбранный период.

## Error
Показывается ошибка загрузки с возможностью повторить запрос.

## Filter changed
Показывается промежуточное состояние пересчета после смены периода или фильтра.

## Правила
Карточки и диаграммы сохраняют стабильные размеры во всех состояниях.
