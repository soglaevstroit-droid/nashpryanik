# UI — 30 Общая аналитика

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Общая аналитика показывает суммарные монеты, суммарное рабочее время и динамику по ключевым разрезам.

## Основная композиция
- Верхняя зона: заголовок «Общая аналитика» и выбор периода.
- Карточка «Суммарные монеты».
- Карточка «Суммарное рабочее время».
- Диаграмма динамики по дням.
- Диаграмма динамики по объектам.
- Диаграмма динамики по сотрудникам.

## Диаграммы
- Динамика по дням отображается линейным графиком или столбцами.
- Объекты отображаются списком или горизонтальными столбцами.
- Сотрудники отображаются рейтингом с возможностью открыть детализацию.

## Запрещено
- показывать финансовые решения как уже принятые;
- перегружать первый экран мелкими показателями;
- использовать цвета вне Design System;
- скрывать выбранный период.
