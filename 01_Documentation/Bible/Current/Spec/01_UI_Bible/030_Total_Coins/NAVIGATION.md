# NAVIGATION — 30 Общая аналитика

## Вход на экран
Открывается из нижнего меню финансиста или по нажатию на главный показатель на главной финансиста.

## Выход
- Назад на главную финансиста.
- Открыть аналитику сотрудника.
- Открыть AI рекомендации.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Рекомендации
- Профиль

## Переходы
- Нажатие на строку сотрудника открывает аналитику сотрудника.
- Нажатие на объект применяет фильтр по объекту.
- Смена периода остается внутри текущего экрана.
