# STATES — 02 Уже на работе?

## Основные состояния
- Loading
- Empty
- Ready
- Error
- Offline / слабая сеть
- Photo required, если действие требует фото
- Disabled, если действие недоступно

## Визуальные правила состояний
- Активное действие — оранжевое.
- Успех — зеленый.
- Ошибка — красная.
- Недоступное — серое / черно-белое.
