# VALIDATION — 07 Выполнение этапа

## Проверки данных
- Не выполнять действие без `employee_id`.
- Не выполнять фото-действие без файла фото.
- Не давать взять задачу, если у сотрудника уже есть задача в работе.
- Не давать взять назначенную задачу чужому сотруднику.
- Не закрывать этап без фото.
- Не показывать действие, если оно недоступно.

## Ошибки
Показывать понятное сообщение и не менять состояние экрана, если API вернул ошибку.
