# STATES — 36 История выплат

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Filter changed
- Payment not found
- Disabled action

## Loading
Показываются скелетоны фильтров и списка выплат.

## Empty
Показывается состояние «Нет выплат за выбранный период».

## Payment not found
Показывается, если детальная запись выплаты недоступна или удалена из доступной области просмотра.

## Error
Показывается ошибка загрузки истории с действием повтора.

## Offline
Показываются последние доступные данные, если они сохранены локально, с отметкой о возможной неактуальности.

## Правила
Состояния не должны позволять редактировать историю выплат.
