# UI — 36 История выплат

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
История выплат показывает утвержденные выплаты и их ключевые параметры.

## Основная композиция
- Верхняя зона: заголовок «История выплат».
- Фильтры: объект, сотрудник, период.
- Список выплат.
- Карточка выплаты: сотрудник, объект, период, начислено, рекомендация ИИ, премия, итог, кто утвердил, дата выплаты.
- Пустое состояние при отсутствии выплат.

## Список выплат
- По умолчанию записи сортируются по дате выплаты по убыванию.
- Каждая запись открывает детальный просмотр.
- Итоговая сумма визуально отделена от рекомендации ИИ.

## Запрещено
- редактировать историю выплат напрямую;
- удалять записи из истории с этого экрана;
- показывать рекомендацию ИИ как финальное решение;
- использовать цвета вне Design System.
