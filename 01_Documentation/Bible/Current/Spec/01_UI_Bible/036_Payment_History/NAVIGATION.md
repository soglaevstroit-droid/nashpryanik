# NAVIGATION — 36 История выплат

## Вход на экран
Открывается из экрана выплат, карточки сотрудника или нижнего меню финансиста.

## Выход
- Назад к выплатам.
- Открыть карточку сотрудника.
- Открыть общую аналитику.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Нажатие на выплату открывает детальную запись.
- Нажатие на сотрудника открывает карточку сотрудника.
- Фильтры применяются внутри текущего экрана.
- Возврат из детали сохраняет выбранные фильтры.
