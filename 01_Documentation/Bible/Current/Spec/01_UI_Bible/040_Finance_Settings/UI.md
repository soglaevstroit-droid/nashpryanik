# UI — 40 Настройки финансиста

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Настройки финансиста управляют предпочтениями финансового раздела.

## Основная композиция
- Верхняя зона: заголовок «Настройки».
- Блок «Период аналитики по умолчанию».
- Блок «Отображение монет».
- Блок «Пороги отклонений».
- Блок «Правила премий».
- Блок «Уведомления».
- Блок «Фильтры объектов».
- Переключатель «Режим только просмотр».
- Действие сохранения.

## Контролы
- Период выбирается из утвержденных вариантов.
- Отображение монет настраивается через переключатель или список.
- Пороги отклонений задаются числовыми полями.
- Уведомления настраиваются через переключатели.
- Фильтры объектов выбираются списком доступных объектов.

## Запрещено
- создавать новые стили настроек;
- использовать другие цвета;
- добавлять настройку автоматического утверждения выплат ИИ;
- скрывать режим только просмотр.
