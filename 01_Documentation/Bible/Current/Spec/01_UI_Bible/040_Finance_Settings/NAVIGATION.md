# NAVIGATION — 40 Настройки финансиста

## Вход на экран
Открывается из профиля финансиста, финансовых отчетов или системного меню.

## Выход
- Назад к предыдущему экрану.
- Открыть главную финансиста.
- Открыть финансовые отчеты.
- Открыть уведомления финансиста.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Сохранение оставляет пользователя на экране настроек.
- Отмена возвращает на предыдущий экран без сохранения.
- Переход назад при несохраненных изменениях требует подтверждения.
