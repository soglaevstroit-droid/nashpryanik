# 40 — Настройки финансиста

## Назначение
Экран настроек финансиста: период аналитики по умолчанию, отображение монет, пороги отклонений, правила премий, уведомления, фильтры объектов и режим только просмотр.

## Роль
Финансист / настройки.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Настройки меняют отображение и правила работы финансового раздела в пределах разрешенной бизнес-логики, но не дают ИИ право автоматически утверждать выплаты.
