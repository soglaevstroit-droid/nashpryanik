# LOGIC — 40 Настройки финансиста

## Основная логика
Экран сохраняет настройки финансового раздела для пользователя или роли финансиста в рамках утвержденной бизнес-логики.

## Настройки
- Период аналитики по умолчанию определяет стартовый период на аналитических экранах.
- Отображение монет определяет формат показа монет.
- Пороги отклонений задают границы для уведомлений и аналитики.
- Правила премий задают допустимые параметры рекомендаций и проверок.
- Уведомления определяют, какие события показывать финансисту.
- Фильтры объектов ограничивают стандартную область просмотра.
- Режим только просмотр отключает действия, которые создают решения по выплатам.

## Правила премий
- Правила премий не должны противоречить утвержденной бизнес-логике.
- Изменение правил не пересчитывает утвержденные выплаты задним числом.
- ИИ может использовать правила для рекомендации, но не для автоматического утверждения.

## Режим только просмотр
- В режиме только просмотр финансист видит данные, но не подтверждает выплаты.
- Режим не скрывает историю и аналитику.
- Режим не меняет начисленные монеты.

## Ограничения ИИ
Ни одна настройка не должна давать ИИ право автоматически утверждать выплаты.
