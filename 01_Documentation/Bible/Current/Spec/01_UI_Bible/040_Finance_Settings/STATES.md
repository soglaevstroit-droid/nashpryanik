# STATES — 40 Настройки финансиста

## Состояния
- Loading
- Ready
- Error
- Offline
- Saving
- Saved
- Validation error
- Conflict
- Disabled action

## Loading
Показываются скелетоны блоков настроек.

## Saving
Показывается во время сохранения настроек.

## Saved
Показывается подтверждение успешного сохранения.

## Validation error
Показывается при некорректных значениях порогов, правил или фильтров.

## Conflict
Показывается, если настройки изменились в другой сессии.

## Правила
Состояния не должны скрывать запрет на автоматическое утверждение выплат ИИ.
