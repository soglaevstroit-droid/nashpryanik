# STATES — 37 Аналитика эффективности

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Filter changed
- Partial data
- Disabled action

## Loading
Показываются скелетоны фильтров, карточек и списков.

## Empty
Показывается состояние «Нет данных эффективности за выбранный период».

## Partial data
Показывается, если не хватает плановых ожиданий или части данных по этапам.

## Error
Показывается ошибка загрузки с действием повтора.

## Правила
Состояния не должны превращать рекомендации ИИ в автоматические решения.
