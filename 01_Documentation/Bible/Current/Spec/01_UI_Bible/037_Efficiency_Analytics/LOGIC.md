# LOGIC — 37 Аналитика эффективности

## Основная логика
Экран рассчитывает эффективность сотрудников за выбранный период на основе подтвержденных задач, этапов, рабочего времени и монет.

## Показатели
- Эффективность сотрудников — сводный показатель по факту выполнения работ.
- Сравнение факта и ожиданий — сопоставление реального результата с планом или расчетной нормой.
- Монеты — подтвержденные начисления за период.
- Рабочее время — подтвержденные рабочие смены.
- Закрытые задачи — завершенные задачи за период.
- Скорость закрытия этапов — среднее время завершения этапов.
- Отклонения — заметная разница между фактом и ожиданиями.
- Рекомендации ИИ — объяснимые подсказки для финансиста.

## Отклонения
- Положительное отклонение показывает результат выше ожиданий.
- Отрицательное отклонение показывает результат ниже ожиданий.
- Отклонение не является автоматическим основанием для премии или штрафа.

## Рекомендации ИИ
- ИИ анализирует эффективность и может рекомендовать проверить премию, нагрузку или данные.
- ИИ не утверждает выплаты.
- ИИ не меняет правила премий.
- Финальное решение принимает финансист.

## Правила финансиста
- Финансист использует экран для анализа и проверки.
- Переход в сотрудника сохраняет выбранный период и фильтры.
