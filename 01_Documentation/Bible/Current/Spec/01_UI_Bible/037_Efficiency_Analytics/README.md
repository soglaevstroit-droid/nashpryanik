# 37 — Аналитика эффективности

## Назначение
Экран аналитики эффективности сотрудников: факт, ожидания, монеты, рабочее время, закрытые задачи, скорость этапов, отклонения и рекомендации ИИ.

## Роль
Финансист / финансовая аналитика.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Экран показывает эффективность и отклонения, но не меняет начисления, задачи или выплаты.
