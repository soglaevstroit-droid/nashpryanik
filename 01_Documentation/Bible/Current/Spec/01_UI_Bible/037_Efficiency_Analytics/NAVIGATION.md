# NAVIGATION — 37 Аналитика эффективности

## Вход на экран
Открывается из общей аналитики финансиста или нижнего меню раздела аналитики.

## Выход
- Назад к общей аналитике.
- Открыть карточку сотрудника.
- Открыть AI рекомендации.
- Открыть финансовые отчеты.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Нажатие на сотрудника открывает карточку сотрудника.
- Нажатие на рекомендацию ИИ открывает детальное объяснение.
- Смена фильтров остается внутри текущего экрана.
