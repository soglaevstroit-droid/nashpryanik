# UI — 37 Аналитика эффективности

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Аналитика эффективности показывает вклад сотрудников и отклонения от ожиданий.

## Основная композиция
- Верхняя зона: заголовок «Эффективность» и выбранный период.
- Фильтры: объект, сотрудник, период.
- Карточка общей эффективности.
- Сравнение факта и ожиданий.
- Показатели: монеты, рабочее время, закрытые задачи.
- Скорость закрытия этапов.
- Блок отклонений.
- Блок рекомендаций ИИ.

## Визуализация
- Сравнение факта и ожиданий отображается диаграммой или парными показателями.
- Отклонения показываются списком с приоритетом.
- Рекомендации ИИ визуально отделяются от фактических показателей.

## Запрещено
- использовать новые цвета для рейтингов;
- показывать рекомендацию ИИ как обязательное решение;
- редактировать начисления с экрана;
- перегружать экран управлением задачами.
