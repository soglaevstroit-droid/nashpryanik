# UI — 15 Главная прораба / Сотрудники

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Прораб видит список сотрудников, статус каждого, текущую задачу, прогресс, статистику и монеты.

## Запрещено
- добавлять лишние дашборды;
- перегружать экран аналитикой;
- менять нижнюю навигацию без причины;
- использовать разные стили карточек.
