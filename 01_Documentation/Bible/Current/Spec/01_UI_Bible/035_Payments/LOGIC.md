# LOGIC — 35 Выплаты

## Основная логика
Экран позволяет финансисту принять решение по выплате на основании начисленных монет и рекомендации ИИ.

## Показатели
- Начисленные монеты — подтвержденные монеты сотрудника за период.
- Рекомендация ИИ — рекомендованная премия или объяснение, почему премия не предлагается.
- Размер премии — значение, которое вводит или подтверждает финансист.
- Комментарий — пояснение финансиста к решению.
- Итоговая сумма — начисленные монеты плюс премия.

## Рекомендация ИИ
- ИИ может предложить премию, если оценка стоимости работы выше начисленных монет.
- ИИ не создает выплату.
- ИИ не утверждает выплату.
- Рекомендация не подставляется как финальное решение без действия финансиста.

## Решение финансиста
- Финансист может оставить премию равной нулю.
- Финансист может указать премию меньше, равную или больше рекомендации ИИ, если бизнес-процесс это допускает.
- При изменении премии требуется комментарий, если правило проекта это требует.
- После подтверждения создается запись для истории выплат.

## Ограничения
- Начисленные монеты не редактируются на этом экране.
- Итоговая сумма не может быть отрицательной.
- Повторное подтверждение одной и той же выплаты не допускается.
