# UI — 35 Выплаты

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Финансист видит данные для выплаты и принимает финальное решение.

## Основная композиция
- Верхняя зона: заголовок «Выплаты», сотрудник и период.
- Блок «Начисленные монеты».
- Блок «Рекомендация ИИ».
- Поле «Размер премии».
- Поле «Комментарий».
- Блок «Итоговая сумма».
- Действие подтверждения решения финансистом.

## Итоговая сумма
- Итоговая сумма складывается из начисленных монет и премии.
- Итог пересчитывается сразу после изменения премии.
- Рекомендация ИИ показывается как подсказка, а не как автоматическое значение.

## Запрещено
- выполнять выплату без подтверждения финансиста;
- автоматически принимать рекомендацию ИИ;
- скрывать комментарий при изменении премии;
- использовать цвета вне Design System;
- менять бизнес-логику начисления монет.
