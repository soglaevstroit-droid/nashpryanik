# 35 — Выплаты

## Назначение
Экран принятия решения по выплате: начисленные монеты, рекомендация ИИ, премия, комментарий и итоговая сумма.

## Роль
Финансист / выплаты.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
ИИ никогда не выполняет выплату самостоятельно. Финальное решение принимает финансист.
