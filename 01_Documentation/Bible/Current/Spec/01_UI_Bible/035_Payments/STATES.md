# STATES — 35 Выплаты

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Recalculating
- Saving
- Saved
- Conflict
- Disabled action

## Loading
Показывается скелетон суммы, рекомендации, полей и итогового блока.

## Empty
Показывается, если нет начисленных монет для выплаты за выбранный период.

## Recalculating
Показывается при пересчете итоговой суммы после изменения премии.

## Saving
Показывается во время сохранения решения финансиста.

## Conflict
Показывается, если выплата за выбранный период уже была создана.

## Правила
Состояния не должны скрывать, что финальное решение принимает финансист.
