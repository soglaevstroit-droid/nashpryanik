# VALIDATION — 35 Выплаты

## Проверки
- `employeeId` обязателен.
- `periodFrom` не может быть позже `periodTo`.
- Начисленные монеты не могут быть отрицательными.
- Рекомендация ИИ не может быть отрицательной.
- Размер премии не может быть отрицательным.
- Итоговая сумма должна равняться начисленным монетам плюс премия.
- Итоговая сумма не может быть отрицательной.
- Комментарий обязателен, если премия отличается от рекомендации ИИ и это требуется бизнес-правилом.
- Нельзя создать повторную выплату за тот же период и объект.

## Отображение
- Суммы отображаются целым числом.
- Ошибки валидации показываются рядом с соответствующим полем.
- Нулевая премия отображается как допустимое решение финансиста.
