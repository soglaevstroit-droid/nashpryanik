# VALIDATION — 33 Диаграмма Ганта

## Проверки
- `periodFrom` не может быть позже `periodTo`.
- `scale` должен быть одним из поддержанных значений.
- `employeeId` должен существовать, если передан.
- `objectId` должен существовать, если передан.
- `durationMinutes` не может быть отрицательным.
- `finishedAt` не может быть раньше `startedAt`.
- `isOverdue` должен соответствовать сравнению фактического окончания и срока.

## Отображение
- Длительность отображается в часах и минутах.
- Отсутствующие даты показываются как неполные данные, а не как ноль.
- Просрочка не должна выглядеть как подтвержденная выплата или штраф.
