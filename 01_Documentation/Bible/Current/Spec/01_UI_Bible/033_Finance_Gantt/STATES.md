# STATES — 33 Диаграмма Ганта

## Состояния
- Loading
- Ready
- Empty
- Error
- Offline
- Refreshing
- Filter changed
- Scale changed
- Partial data
- Disabled action

## Loading
Показывается скелетон фильтров, строк и временной шкалы.

## Empty
Показывается состояние «Нет задач за выбранный период».

## Partial data
Показывается, если у задачи или этапа нет полного времени начала или окончания.

## Error
Показывается ошибка загрузки диаграммы с действием повтора.

## Правила
Смена фильтра или масштаба не должна ломать структуру строк и временной шкалы.
