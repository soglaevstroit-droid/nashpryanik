# UI — 33 Диаграмма Ганта

## Стиль
- Architect Minimal.
- Светлый фон.
- Мягкие карточки.
- Фирменный оранжевый `#F07A00`.
- Зеленый только для статуса «На работе» / успеха.
- Серый / черно-белый для недоступного.

## Экран
Диаграмма Ганта отображает работы сотрудников во времени.

## Основная композиция
- Верхняя зона: заголовок «Диаграмма Ганта» и выбранный период.
- Фильтры: объект, сотрудник, период.
- Переключатель масштаба временной шкалы.
- Левая колонка: сотрудники, задачи и этапы.
- Правая область: временная шкала с началом, окончанием, длительностью и статусом.
- Маркеры просрочек.

## Временная шкала
- Масштаб поддерживает день, неделю и месяц.
- Начало и окончание задачи отображаются по фактическим данным, если они есть.
- Плановые сроки могут отображаться тонкой линией, если доступны в данных.
- Просрочка выделяется аккуратно, без новых цветов вне Design System.

## Запрещено
- создавать новые стили диаграмм;
- использовать другие цвета;
- редактировать задачи с диаграммы;
- скрывать просрочки;
- превращать экран в планировщик прораба.
