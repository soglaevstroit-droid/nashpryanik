# 33 — Диаграмма Ганта

## Назначение
Экран финансовой диаграммы Ганта для просмотра сотрудников, задач, этапов, сроков, длительности, статусов и просрочек.

## Роль
Финансист / финансовая аналитика.

## Статус
APPROVED REFERENCE REQUIRED.

## Требование
Сохранить визуальный стиль утвержденных референсов минимум на 95%.

## Главное правило
Диаграмма Ганта показывает фактическую картину выполнения работ и не меняет задачи, монеты или выплаты.
