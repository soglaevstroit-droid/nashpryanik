# LOGIC — 33 Диаграмма Ганта

## Основная логика
Экран строит временную диаграмму по сотрудникам, задачам и этапам за выбранный период.

## Отображаемые сущности
- Сотрудник.
- Задача.
- Этап задачи.
- Время начала.
- Время окончания.
- Длительность.
- Статус.
- Просрочка.

## Фильтры
- Фильтр по объекту ограничивает задачи и этапы выбранным объектом.
- Фильтр по сотруднику показывает задачи, где сотрудник является исполнителем или участником.
- Фильтр по периоду показывает пересечения задач и этапов с выбранным диапазоном.
- Масштаб временной шкалы меняет только визуализацию, а не данные.

## Расчет длительности
- Длительность считается между временем начала и временем окончания.
- Для активной задачи окончанием считается текущее время.
- Если начало или окончание отсутствует, элемент отображается в состоянии неполных данных.

## Просрочки
- Просрочка показывается, если фактическое окончание позже планового срока.
- Для активной задачи просрочка показывается, если текущий момент позже планового срока.
- Просрочка не меняет начисленные монеты автоматически.

## Правила финансиста
- Финансист анализирует сроки и вклад сотрудников.
- Финансист не редактирует задачи, этапы и исполнителей на этом экране.
