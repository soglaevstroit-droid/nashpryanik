# NAVIGATION — 33 Диаграмма Ганта

## Вход на экран
Открывается из общей аналитики финансиста или из нижнего меню раздела аналитики.

## Выход
- Назад к общей аналитике.
- Открыть карточку сотрудника.
- Открыть историю выплат.
- Открыть профиль.

## Нижнее меню финансиста
- Главная
- Аналитика
- Выплаты
- Профиль

## Переходы
- Нажатие на сотрудника открывает карточку сотрудника.
- Нажатие на задачу открывает доступный просмотр задачи.
- Смена фильтров остается внутри текущего экрана.
- Смена масштаба остается внутри текущего экрана.
