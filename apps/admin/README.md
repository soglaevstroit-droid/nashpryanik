# apps/admin

Назначение: зарезервированное место для будущих административных или операционных поверхностей.

Ограничение: наличие каталога не вводит роль администратора, admin UI, admin routes или новые права доступа. Любая такая функциональность возможна только после появления роли и сценариев в фундаментальной документации.

На Sprint 001 каталог создается как часть монорепозитория и инфраструктурной структуры. Прикладной код не создается.
